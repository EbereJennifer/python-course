1) Welche Diagramme werden für Datenmodellierung benutzt?

a) UML-Diagramme  
b) ER-Diagramme 
c) Struktogramme 
d) Flussdiagramme

answer -> b
ER-Diagramme (Entity-Relationship-Diagrams) are a way to visually model a database.
a) UML-Diagramme Used for software design (classes, objects, behavior), not mainly for database modeling
Unified Modeling Language diagrams can be used for data modeling.
Especially class diagrams:
represent entities (classes)
show attributes (fields)
show relationships between them

b) ER-Diagramme
Entity-Relationship Diagram is specifically designed for data modeling.
Shows:
entities (e.g., Customer, Order)
attributes (name, ID, etc.)
relationships (e.g., “Customer places Order”)

👉 This is the classic and most important data modeling diagram
